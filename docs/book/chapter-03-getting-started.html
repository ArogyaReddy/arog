üìã Prerequisites

  • VS Code (recommended, but not required)

üéØ The 8-Step Installation

1 Clone or Download AROG

Get the AROG framework to your local machine:

git clone https://github.com/YourOrg/AROG.git
cd AROG

Alternative: Download the ZIP file from GitHub and extract it.

2 Install Dependencies

Install all required packages with a single command:

npm install

This installs everything: Jest, Playwright, ESLint, Prettier, TypeScript, and more. Takes about 2-3 minutes on a good connection.

3 Run Automated Setup

Let AROG configure itself automatically:

npm run arog:setup

This magical command:

  • ‚úÖ Installs Playwright browsers
  • ‚úÖ Creates directory structure
  • ‚úÖ Validates all configurations
  • ‚úÖ Runs a health check
  • ‚úÖ Generates setup report
4 Validate Installation

Verify that everything is configured correctly:

npm run arog:validate

You should see a beautiful green checkmark next to each component:

5 Create GitHub Repository

Time to connect AROG to GitHub. First, create a new repository:

  • Go to GitHub.com
  • Click the "+" icon ‚Üí "New repository"
  • Name it (e.g., "my-arog-project")
  • Keep it public or private (your choice)
  • Don't initialize with README (AROG has one)
  • Click "Create repository"
6 Push to GitHub

Connect your local AROG to the GitHub repository:

git remote add origin https://github.com/YourUsername/your-repo.git
git branch -M main
git add .
git commit -m "feat: Initialize AROG automation framework"
git push -u origin main

Watch the magic: As soon as you push, GitHub Actions start running automatically!

7 Enable GitHub Actions

GitHub might ask for permission to run workflows. Here's how:

  • Go to your repository on GitHub
  • Click the "Actions" tab
  • Click "I understand my workflows, go ahead and enable them"
  • Watch your first automated workflow run!

You'll see 8 workflows ready to go:

8 Activate @arog Agent

If you have GitHub Copilot, you can talk to @arog right now:

  1. Open VS Code
  2. Open your AROG project folder
  3. Open GitHub Copilot Chat (Cmd/Ctrl + Shift + I)
  4. Type: @arog hello
  5. Watch @arog respond!
@arog hello
# @arog introduces itself

@arog what can you do?
# @arog lists its capabilities
